# Appliance Inventory Web App
[Link to Demo!](https://appliance-inventory-web-mgkh.vercel.app)

A full-stack web application for managing a personal appliance inventory. Built with React, Node.js, Express, and PostgreSQL.

## Features
- **User Authentication:**
Register and log in with secure hashed passwords and JWT token-based sessions
- **Appliance Management:**
Add, edit, and delete appliances with type, brand, and price tracking
- **Search and Filter:**
Filter appliances by brand or maximum price
- **Live Currency Conversion:**
Toggle between 8 currencies (USD, CAD, EUR, GBP, JPY, AUD, CHF, CNY) using the ExchangeRate API
- **Persistent Storage:**
All data saved to a PostgreSQL database per user account


## Frontend
- React
- Tailwind CSS

## Backend
- Node.js
- Express.js
- PostgreSQL
- JWT Authentication
- bcrypt password hashing

## External API
- ExchangeRate API (used for the live currency conversion)
